# Docs

This folder contains all stored documents, organized by type.

## Subfolders

| Folder | Contents |
|--------|----------|
| [`PDFs/`](PDFs/) | PDF documents of all kinds |
| [`eBooks/`](eBooks/) | eBooks in EPUB, MOBI, AZW, or PDF format |
| [`Presentations/`](Presentations/) | Slides and presentations (PPTX, KEY, ODP) |
| [`Spreadsheets/`](Spreadsheets/) | Data files and spreadsheets (XLSX, CSV, ODS) |
| [`Other/`](Other/) | Word documents, text files, and anything else |
